# Changelog

## Unreleased

- 将 Fail2ban 状态页升级为完整管理中心，支持安装、启停、重启、开机策略、Jail 统计、日志、配置重载与解除误封
- 将 UFW 升级为可交互防火墙管理器，支持启停、重载、日志级别、TCP/UDP、端口范围、来源 IP/CIDR 和编号规则删除
- 应用概览升级为应用服务管理页，可直接控制 Docker、Web、缓存、数据库与已安装的 3x-ui 服务
- Docker 增加 Daemon 服务管理入口；运行中和失败服务清单可直接进入详情与生命周期操作
- 服务生命周期操作增加执行后验证与统一反馈，避免命令成功返回但目标状态未生效
- 时间同步增加启用、禁用、同步状态和实际同步服务管理，系统仪表盘增加基础防护状态
- BBR 增加状态面板和仅针对工具托管配置的安全恢复入口
- 优化信息面板边框、彩色状态菜单项、动态操作提示与安全中心信息架构
- 将 Swap 单向创建升级为带状态所有权记录的生命周期管理，支持即时启停和仅删除工具托管文件
- 将 systemd Timer 从静态列表升级为调度管理页，展示触发信息并支持启停、开机策略和立即运行关联任务
- Journal 增加文件一致性验证、持久化状态、启动记录统计以及按时间或容量清理
- Docker Compose 增加项目上下文识别、服务状态、日志、镜像拉取、`up`、停止、重启和不删除卷的 `down`
- 增加 `serverctl swap`、`timer`、`journal` 与 `compose` 直接入口
- 将用户会话页升级为账户管理中心，增加本地账户创建、认证锁定状态、sudo 权限保护、SSH 公钥写入与指纹查看
- 进程页增加 PID 下钻、命令与 systemd 归属、线程/FD、打开文件、网络套接字、cgroup、nice 和 TERM/KILL 控制
- 为 root、当前操作账户、最后一个 sudo 成员、PID 1 与工具父子进程建立显式保护边界
- 增加 `serverctl users`、`user USER` 与 `process PID` 直接入口

## 0.2.0

- 接入 Starship、Oh My Posh 与 Spaceship Prompt，提供面向实际登录用户的安装、互斥启用、更新和安全卸载
- 提示符使用独立托管配置块，切换时保留其他引擎与用户配置，并明确 Nerd Font 位于 SSH 客户端的边界
- 软件目录扩展到 160 项，新增终端美化、网络、开发、安全、监控、运行时与服务软件
- 增加 Oh My Zsh 官方 Git 安装器，支持目标用户识别、`.zshrc` 备份、更新、可选默认 Shell 与边界明确的卸载
- 软件中心增加软件源可用性检测；仓库未提供的软件会显示明确状态并禁用安装操作
- CI 改为逐个检查每个 Shell 文件，兼容 ShellCheck 0.9/0.11，并为跨模块全局变量与测试桩补充精确说明
- 功能分支仅运行 pull request 检查，避免 push 与 pull_request 重复执行；GitHub checkout 升级到 Node.js 24 运行时
- 将 CI 拆分为 `repository-files`、`shell` 和 `tests` 三项检查，并强制所有仓库文件归类验证
- 增加 UTF-8、LF、末尾换行、尾随空白、Markdown 内部链接、YAML、SVG XML、Git 属性和发布元数据检查
- 软件目录曾由 66 项扩展到 106 项，增加系统工具、网络、安全、监控、数据库客户端和 OCI 容器工具
- 软件中心增加按分类浏览与仅看已安装视图，避免目录扩充后出现过长的平铺列表
- 增加 DNS 解析器与 A/AAAA/CNAME 记录诊断，并提供 `serverctl dns [域名]` 直接命令
- TLS 证书检查增加主机名/IP 匹配验证和剩余有效期分级预警
- 将系统修改与维护动作拆分到 `src/features/system/settings.sh`，降低系统中心单文件复杂度
- 增加综合系统健康巡检，汇总资源、存储、systemd、软件更新、DNS、HTTPS、时间同步、防火墙和 SSH 状态
- 增加 `serverctl health`、`serverctl updates`、`serverctl storage` 与 `serverctl logs SERVICE` 直接命令
- 将软件包状态页扩展为更新版本清单、保留包、dpkg 审计与缓存摘要
- 将磁盘分析扩展为块设备、空间、inode、只读挂载和已删除文件占用诊断
- 备份中心增加完整性校验、当前配置差异、安全删除、快照大小与文件数量展示
- Docker 增加容器健康检查，并将容器操作升级为包含状态、镜像、端口、挂载、日志、资源和生命周期的详情页
- 服务中心增加关键词查找、描述、子状态、主进程、重启次数、状态时间与依赖关系
