# Changelog

## 0.1.0

- 建立全新的 Debian / Ubuntu 基线，不兼容旧版结构
- 使用八个一层式功能中心替代混乱的多层模块菜单
- 建立一次一个软件的目录与安装流程
- Docker、Caddy 作为单项软件接入官方仓库，不附带防火墙等额外配置
- Docker 安装会检测冲突包并单独确认，同时提示容器端口与 UFW 的边界
- 提供系统仪表盘、环境检查、主机名、时区、Swap、时间同步、软件包健康、磁盘分析和安全清理
- 提供网络诊断、监听端口与进程查询、BBR、IP 优先级、防火墙、SSH 向导和登录事件
- 提供 systemd 服务与 Journal 管理、Docker 管理、操作审计、配置快照与单文件恢复
- 删除 profiles、无人值守确认、软件套餐、重复工具入口、自动更新、TCP 调优档位、修复中心和报告生成
- 重写安装器，升级时原子替换运行目录并拒绝危险目标
- 将运行入口迁至 `bin/`，核心与功能代码迁至 `src/`，数据迁至 `config/`
- 根目录安装器缩减为稳定引导器，完整安装逻辑迁至 `scripts/`
- 增加离线结构、目录、CLI、备份和安全边界测试
