# Changelog

## 0.1.0

- 首个公开基线版本，版本号重新整理为 `0.1.0`
- 首屏 UI 升级为卡片式中文仪表盘，加入边框、分隔线和状态徽标
- 主菜单按基础与系统、网络与安全、服务与运行时、管理分组
- 新增卸载功能：`serverctl uninstall`
- 安装器新增远程卸载：`install.sh --uninstall`
- 支持 `--purge` 在卸载时同时删除日志和备份
- 安装器下载源码时默认静默显示，避免 curl 进度条污染首次菜单
- 安装完成后进入菜单前自动清屏
- 80/443 端口检测改为短摘要，例如 `caddy(tcp)`
- 新增系统设置中心：主机名、时区、Locale、hosts 修复、Swap、管理员用户、时间同步、自动更新
- 软件安装中心提供基础工具、现代 CLI、编译库、代理节点依赖、备份同步、安全工具等分类
- 支持 `minimal`、`proxy`、`docker`、`web`、`dev`、`full` profiles
