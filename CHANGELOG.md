# Changelog

## 0.1.0

- 首个公开基线版本，版本号重新整理为 `0.1.0`
- 首屏 UI 升级为卡片式中文仪表盘，加入边框、分隔线和状态徽标
- 面板渲染升级为完整左右边框，自动清洗多行状态值，避免输出打断竖线
- 修正中文双宽字符导致的右边框错位问题
- 功能菜单升级为控制台式分组面板，支持 `00/01` 这类两位编号输入
- 精简功能入口标题，移除冗余说明短句
- README 重写为正式项目文档，移除对话式说明
- 子界面菜单统一升级为面板 UI，包括基础初始化、系统设置、软件中心、网络、防火墙、Docker、Web、数据库、工具、修复等
- 新增 `serverctl software` 和 `serverctl firewall` 直接入口
- 系统检测增加会话缓存，重复进入模块不再反复执行完整检测
- GitHub 连通性检查默认缩短超时，减少网络异常时的卡顿
- 软件源刷新增加会话缓存，避免同一轮使用中重复 `apt update`
- 主菜单按基础与系统、网络与安全、服务与运行时、管理分组
- 新增卸载功能：`serverctl uninstall`
- 安装器新增远程卸载：`install.sh --uninstall`
- 卸载时支持交互选择范围：仅入口、入口+安装目录、入口+安装目录+日志、全部删除
- 支持 `--purge` 在无人值守卸载时同时删除日志和备份
- 安装器下载源码时默认静默显示，避免 curl 进度条污染首次菜单
- 安装完成后进入菜单前自动清屏
- 80/443 端口检测改为短摘要，例如 `caddy(tcp)`
- 新增系统设置中心：主机名、时区、Locale、hosts 修复、Swap、管理员用户、时间同步、自动更新
- 软件安装中心提供基础工具、现代 CLI、编译库、代理节点依赖、备份同步、安全工具等分类
- 支持 `minimal`、`proxy`、`docker`、`web`、`dev`、`full` profiles
