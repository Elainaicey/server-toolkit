# Changelog

## 0.2.0

- 主菜单重新分类，加入状态摘要和更清晰的模块入口
- 新增系统设置中心：主机名、时区、Locale、hosts 修复、Swap、管理员用户、时间同步、自动更新
- 软件安装中心升级为分类菜单，新增现代 CLI、编译库、代理节点依赖、备份同步、安全工具等类别
- 系统检测新增建议区，提示 APT 异常、端口冲突、IPv6-only、GitHub 连通性、防火墙状态
- 新增 `dev` 和 `full` profiles
- 快速初始化改为菜单式，不再进入模块就立即刷新软件源

## 0.1.1

- 安装器默认在安装后打开中文交互菜单
- 新增 `--menu` / `--no-menu`
- 大部分用户可见提示改为中文
- 交互输入优先从 `/dev/tty` 读取，兼容 `bash <(curl ...)` 和 `curl | bash`

## 0.1.0

- 初始版本
- 支持主菜单、系统检测、报告生成
- 支持基础初始化、网络/BBR、SSH、防火墙、Docker、Web、数据库、监控、修复、回滚模块
- 支持 `minimal`、`proxy`、`docker`、`web` profiles
- 支持 GitHub 远程安装器 `install.sh`
